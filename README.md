# Reseller App Form Flow.

## Intro
A simple form flow for a reseller app.  

## How To Use
Open the terminal in the project folder and run:
```sh
  npm install
```
or run yarn
```sh
  yarn
```
The dependies will be installed on your local machine.  
Then:
```sh
  npm start
```
or run yarn
```sh
  yarn start
```
